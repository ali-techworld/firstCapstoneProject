Core Curriculum

  • User Research

    This program focuses on teaching participants how to conduct effective user research, gather insights, and validate design decisions through usability testing.

    This program delves into the specifics of designing interactive interfaces that provide seamless and intuitive user interactions.

    This program focuses on the visual aspects of design, teaching participants about color theory, typography, layout, and other design elements.
